# NER-Microservice
NER microservice template

## Docker commands
- `docker build . -t <tag_name>`
- `docker image ls` # list the docker ids
- `docker run -p 127.0.0.1:8080:8080 <image_id>`
- to remove a docker image `docker rmi <image_id>`
